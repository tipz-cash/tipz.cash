# Grant Research

> Research notes on funding opportunities for TIPZ.
> Goal: Identify realistic grant programs and ask amounts.

---

## Zcash Foundation (PRIMARY TARGET)

### Program Overview

**Website:** https://zcashcommunitygrants.org/
**Application Portal:** https://zcashcommunitygrants.org/submit
**Grant Cycles:** Rolling applications, reviewed monthly by Zcash Community Grants Committee
**Typical Award Size:** $10K-$100K (larger grants require more milestones)

### Application Requirements

- [x] Clear problem statement (privacy focused)
- [x] Technical approach (shielded addresses)
- [x] Milestone-based deliverables (3-6 months)
- [x] Budget breakdown with justification
- [x] Team background
- [ ] Demo video or working prototype
- [ ] Community engagement (forum post)

### Relevant Past Grants (Examples)

| Project | Amount | Date | Notes |
|---------|--------|------|-------|
| Zashi Wallet | $500K+ | 2023-24 | Mobile wallet (larger team) |
| Zcash Media | $100K | 2022 | Educational content |
| ZecWallet Lite | $50K | 2021 | Desktop wallet |
| Zcash Shielded Assets | $75K | 2022 | Protocol development |

*Source: forum.zcashcommunity.com grant discussions*

### TIPZ Fit

**Strengths:**
- Increases shielded transaction usage (every tip = 1 shielded tx)
- Real-world privacy use case (creator income)
- Non-custodial (aligns with Zcash ethos)
- Creator economy focus (new audience for Zcash)
- Cross-chain bridge (ETH users discover ZEC)
- Working MVP (not vaporware)

**Potential Concerns:**
- Solo founder (mitigate: show shipping velocity)
- NEAR dependency (mitigate: direct ZEC fallback)
- Creator adoption unproven (mitigate: seed 10+ before applying)

### Recommended Ask

**Amount:** $40,000 USD (payable in ZEC)

**Rationale:**
- Reasonable for solo founder with working MVP
- Covers 6 months runway to reach sustainability
- Milestone-based reduces risk for ZCG
- Comparable to similar ecosystem tools

See `zcash-grant-application.md` for full application draft.

## NEAR Ecosystem (SECONDARY TARGET)

### Program Overview

**Website:** https://near.org/ecosystem/grants
**Application Portal:** https://near.org/ecosystem/grants (Apply Now button)
**Grant Cycles:** Rolling, reviewed by NEAR Foundation
**Typical Award Size:** $5K-$50K for integrations

### Requirements

- [x] NEAR protocol integration (using NEAR Intents)
- [x] Technical documentation
- [ ] Mainnet deployment
- [ ] User metrics post-launch

### Relevant Past Grants

| Project | Amount | Date | Notes |
|---------|--------|------|-------|
| Rainbow Bridge | $100K+ | 2022 | Cross-chain bridge |
| Various DeFi | $25-50K | 2023 | Integration grants |
| Intents Protocol | Funded | 2024 | Core infrastructure |

### TIPZ Fit

**Strengths:**
- Real-world NEAR Intents implementation
- Cross-chain routing showcase (ETH -> ZEC via NEAR)
- Privacy-preserving use case (unique in NEAR ecosystem)
- Case study potential for Intents docs

**Potential Concerns:**
- Privacy focus may not resonate with NEAR culture
- Small scale compared to typical NEAR grants
- ZEC as destination chain (not NEAR native)

### Recommended Ask

**Amount:** $15,000 USD (in NEAR)

**Rationale:**
- Covers infrastructure and gas costs
- Smaller ask = faster approval
- Position as "Intents showcase" not privacy tool
- Apply AFTER Zcash grant (less urgent)
